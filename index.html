UMA BREVE INTRODUÇÃO SOBRE O PROJETO

Olá, meu nome é Juan! Esse é o meu primeiro projeto teste criado à partir das noções do módulo do curso de JavaScript e TypeScript destinado ao HTML5 e CSS3 para iniciantes, apenas para ter noção das funcionalidades básicas da linguagem. Sem nenhum conteúdo expressivo, apenas uma Landing Page para testar a estrutura e as funcionalidades.

QUAIS AS FUNCIONALIDADES?

Alguns testes realizados para entender a interação de elementos dentro do HTML e CSS, testes com validação, flexbox, margin, padding, viewport, headings, id, class, menus, tabelas, semântica, especificidade e herança, box model, display block e float, position, diferentes unidades de medida e SVG. Sem um grande aprofundamento e detalhamento de cada área, mas para entender de uma forma mais básica, e a princípio para ter noções básicas e intermediárias sobre o HTML e CSS unido ao JavaScript (apenas para SVG).

Dessa forma, mesclando os ensinamentos obtidos nas aulas junto com outras pesquisas para auxiliar, consegui criar algo que não fosse uma cópia de códigos, mas um projeto com elementos personalizados e criados em cima de um conhecimento assimilado.
